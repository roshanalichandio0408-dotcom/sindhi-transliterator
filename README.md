# Romanized Sindhi to Perso-Arabic Sindhi Transliterator (FYP 2026)

This project is a Final Year Project (FYP) that converts (Romanized Sindhi text into Perso-Arabic Sindhi script) using deep learning transformer models.

The system is designed to help bridge the gap between informal Roman Sindhi writing (commonly used in chats and social media) and the official Perso-Arabic Sindhi script.

---

 Project Objective

The main goal of this project is to:

- Convert **Romanized Sindhi → Perso-Arabic Sindhi**
- Improve accessibility of Sindhi digital communication
- Preserve correct script representation using AI-based models

 Features

- 🔤 Roman Sindhi to Perso-Arabic Sindhi conversion
- 🤖 Transformer-based sequence-to-sequence models
- 🌐 Web-based user interface
- ⚡ Real-time text transliteration
- ☁️ Model hosted on Hugging Face Hub
- 📡 API-based inference system


 Models Used

This project uses fine-tuned transformer models for transliteration:

- mBART (Multilingual BART)
- mT5 (Multilingual Text-to-Text Transformer)
- Custom fine-tuned checkpoint for Roman → Sindhi script mapping
