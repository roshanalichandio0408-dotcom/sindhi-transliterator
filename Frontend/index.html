About the Project

Preserving a Language
Through Technology

Sindhi is a low-resource language spoken by millions. This tool bridges the gap between Roman transliteration (commonly used online) and the traditional Perso-Arabic script, using state-of-the-art multilingual transformer models.

🔄

Bidirectional

Convert seamlessly between Roman and Perso-Arabic Sindhi script with high accuracy.

🤖

Dual Model Ensemble

Two fine-tuned transformer models (mBART & mT5) run in parallel for side-by-side comparison.

📜

History Tracking

Every conversion is saved to your account so you can revisit and review past transliterations.

🔒

Secure Auth

Firebase Authentication with JWT tokens keeps your account and history protected.

Models

Two Models, One Result

Both models are fine-tuned on the same Sindhi dataset and run simultaneously so you can compare outputs.

Meta AI

mBART-50

facebook/mbart-large-50-many-to-many-mmt fine-tuned for Sindhi. Achieves ~97.80 character-level BLEU, excelling at character-accurate output.

Google

mT5-small

google/mt5-small fine-tuned on 9,800+ transliteration pairs. Achieves ~91.86 BLEU and ~98% character accuracy on the test set.
